## 🪪 Delphy + Delphy Agent

> **The @identity for the agentic web, and the desktop client that drives it.**

**Delphy** gives any business, person, or service a machine-readable `@identity` that AI agents can discover and act on. No scraping, no setup.

| Layer | Detail |
|---|---|
| 🆔 Identity | `delphy.json` manifest + agent-readable `skill.md` |
| 🔌 MCP | Published server (`delphy-mcp` on npm) |
| 🔎 Discovery | Search API + OpenStreetMap import |
| ☁️ Stack | Next.js + Supabase, OpenAPI 3.1, OAuth |

🔗 **[delphy.network](https://delphy.network)**

**Delphy Agent** is a desktop hub that drives 8 LLM providers and agent CLIs through one interface, extended with MCP plugins. Built with Tauri v2 + React 19.

🔗 **[github.com/fermartz/delphy-agent](https://github.com/fermartz/delphy-agent)**
